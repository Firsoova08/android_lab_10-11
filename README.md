# Student List App

**Student List** — это учебное Android-приложение, демонстрирующее применение принципов Material Design на практике. Приложение отображает список студентов, предоставляя возможность просматривать их аватарки, имена, возраст и юмористические описания. Интерфейс поддерживает светлую и тёмную темы, а также включает анимацию раскрытия карточек.

## Цель работы

Изучить и применить на практике основные принципы Material Design при разработке Android-приложений с использованием Jetpack Compose. В ходе работы были освоены:
- Настройка цветовых схем (Color Scheme) для светлой и тёмной тем.
- Работа с типографикой (Typography) и подключение пользовательских шрифтов.
- Применение форм (Shape) для создания визуальной иерархии.
- Создание адаптивного интерфейса с использованием `Scaffold` и `TopAppBar`.
- Реализация анимаций и интерактивных элементов (раскрывающиеся карточки).

## Функциональность

Приложение обладает следующим функционалом:

*   **Отображение списка студентов:** Список из 10 студентов, каждый представлен в виде карточки.
*   **Аватарки студентов:** Круглые аватарки, корректно обрезанные с помощью `ContentScale.Crop`.
*   **Информация о студенте:** Имя и возраст отображаются в каждой карточке.
*   **Раскрывающееся описание:** При нажатии на иконку (▼ / ▲) в карточке плавно раскрывается или скрывается юмористическое описание студента.
*   **Поддержка светлой и тёмной темы:** Приложение автоматически переключает цветовую схему в зависимости от настроек системы.
*   **Верхняя панель приложения:** Содержит логотип и название приложения, оформлена в стиле Material Design.

## Технологии

В проекте были использованы следующие технологии и библиотеки:

*   **Язык:** Kotlin
*   **UI Toolkit:** Jetpack Compose
*   **Дизайн-система:** Material Design 3 (Material You)
*   **Набор иконок:** `androidx.compose.material:material-icons-extended` (для иконок ExpandLess/ExpandMore)
*   **Шрифты:** Abril Fatface, Montserrat (загружены из Google Fonts и интегрированы в проект)
*   **Система сборки:** Gradle (Kotlin DSL)
